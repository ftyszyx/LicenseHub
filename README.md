pay for app

# 系统说明

## 背景：

我做了一个软件，想为这个软件做一个付费系统

需求如下：

用户可以在这个系统上买注册码，在我的软件输入后，就会绑定软件安装的设备。

1. 注册码需要有有效期
2. 注册码需要有设备绑定限制
3. 需要提供接口给客户端验证注册码是否有效
4. 系统需要提供支付功能，如支付宝和微信

## 系统：

1. 用户可以购买系统的商品，这个商品对应的是一个注册码，
1. 注册码有有效期、不同有效期的价格不一样。
1. 购买完后，系统会自动生成一个注册码
1. 用户使用此注册码绑定设备后，系统会有记录
1. 注册码会限制绑定设备数量，默认是一个设备。
